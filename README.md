# AI Study Assistant

AI-powered study assistant built with Flutter that helps students learn smarter using AI.

The app can:
- Generate quizzes
- Create AI summaries
- Answer study questions
- Help with quick revision

Built using Flutter + OpenRouter API.

## Installation

Clone the repository:

```bash
git clone https://github.com/tritvika2006-byte/ai-study-assistant.git
```

Go inside project:

```bash
cd ai-study-assistant
```

Install dependencies:

```bash
flutter pub get
```

Run app:

```bash
flutter run
```

---

## API Setup

This project uses OpenRouter API.

Create your own API key from:

https://openrouter.ai/

Then add your API key inside:

```dart
lib/services/gemini_service.dart
```
